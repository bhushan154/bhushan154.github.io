Jira Development Essentials

New to JIRA plugin development? This website will help you with step-by-step tutorials to all the different plugin modules available in JIRA. We will start with simple plugin modules, consider different requirement scenarios which will help you relate to the requirements you take up at your job.

What do I need to know? You need to have experience programming in Java. You should have some experience using Atlassian JIRA and have an understanding of what the tool is used for and how it is used. Experience administering JIRA, is an added benefit.

How do I use this website? If you are new to JIRA plugin development, start from the very first tutorial available on this site. If you are looking for help developing a particular plugin module, you can skip straight to the guide.

Who am I? My name is Bhushan Nagaraj. I have more than 3 years of experience administering JIRA and developing plugins for JIRA. I am completely aware that there is enough material on the Atlassian site to help developers. I just prefer guides with screenshots and every step captured. I have also made an attempt to organize this content to help complete new comers.
